# fourth-order-compact-scheme-for-fractional-Ginzburg-Landau
A fourth-order compact finite difference scheme with BDF2‑θ time discretization and ADI implementation for the two‑dimensional space‑fractional complex Ginzburg–Landau equation. Stability and convergence in L² norm are proved. Numerical examples confirm O(τ² + h⁴) accuracy.

二维空间分数阶复金茨堡‑朗道方程的四阶紧致差分格式，采用 BDF2‑θ 时间离散、紧致有限差分空间离散及 ADI 快速算法，理论证明了 L² 范数下的稳定性和 O(τ² + h⁴) 收敛阶，数值算例验证了方法的高精度。
